# DuggansDingers v13

This build removes the login screen and applies the approved dark sports-intelligence redesign.

## Streamlit deployment
- Repository branch: `main`
- Main file path: `DuggansDingers_v13_exact_redesign/app.py` if uploaded as a folder
- Or `app.py` if the contents are uploaded directly to the repository root

Keep API credentials in Streamlit **Settings → Secrets**. Do not upload `.env` or `.streamlit/secrets.toml`.
